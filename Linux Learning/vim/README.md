## 常用命令

### 12-21
```
	vs *.py  分屏
	ctrl+w l  鼠标移至左屏
	ctrl+w h  鼠标移至右屏

	文本选择
	v v结束 

	d delete
	y copy
	p paste
	ggVG  gg跳到行首  V选中整行  G末行

	x 删除单个字符
	dd 删除整行

	ctrl+R  撤消撤消
	U重做

	#插入文本
	a在当前光标后插入文本
	o可以在当前行下另起一行

	ZZ = :wq


	移动光标
	w 以word为单位移动光标
	$ 移动到行尾

	%
	匹配()


	搜索
	/

	yw复制一个word

	diw删除一个word


	编辑另外一个文件
	vim *py,*py
	:n :N  ctrl+^来回切换

	显示当前正在编辑文件
	:args

	：!cmd 运行cmd后回到文本
```
